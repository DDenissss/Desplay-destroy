        RUSSIAN

1. запутить файл Библиотеки.bat
2. запустить фал compil.bat
4. зайти в папку dist
5. открыть файл Destroy display.exe

Для того что бы закрыть программу нужно открыть файл destroy.bat
или:

1. Открыть диспетчер задач найтипроцесс Destroy display.exe нажать по нему пкм и в открывшемся окне нажать снять задачу
2. Открыть диспетчер задач найтипроцесс Destroy display.exe нажать по нему лкм и нажать клавишу Enter
3. Нажать сочитание клавиш Win + r далее в открывшемся окне вписать cmd затем нажать клавишу Enter после появится окно, туда нужно вписать ( taskkill /f /im "Destroy display.exe" )

        ENGLISH
        
1. Run the Library file.bat
2. Launch the halyard collection.bat
4. go to the distance folder
5. Open the Destroy file display.exe

In order to close the program, you need to open the destroy file.bat
or:

1. Open the Task Manager and destroy the process display.exe click on it pcm and in the window that opens, click remove task
2. Open the Task Manager and destroy the process display.exe click on it lmb and press Enter
3. Press the combination of the Win + r keys, then enter cmd in the window that opens, then press Enter after a window appears, you need to enter (taskkill /f /im "Destroy display.exe " )
